# Port Me

Simple Python Port Scanner built with Python sockets and threading.

![Port Me Screenshot](images/screenshot1.png)

## Features

- Single port scan
- Multi-port scan
- Service detection
- Threading support
- File logging
- Hostname validation
- Port validation
- Scan summary
