# Port Me

Simple Python Port Scanner built with Python sockets and threading.

## Features

- Single port scan
- Multi-port scan
- Service detection
- Threading support
- File logging
- Hostname validation
- Port validation
- Scan summary

## Example
[OPEN] 21 (FTP)
[OPEN] 80 (HTTP)
[OPEN] 25 (SMTP)
[OPEN] 53 (DNS)
Scan complete. Results saved to results.txt

## Requirements

Python 3

## Run

```bash
python3 portme.py
